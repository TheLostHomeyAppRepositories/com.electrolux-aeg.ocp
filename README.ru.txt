Это приложение Homey подключится к
• Платформе идентификации Gigya (принадлежит SAP) через NPM @gslender/gigya API
• API Electrolux OCP (OneConnectedPlatform) на https://api.ocp.electrolux.one/appliance/api/v2
• и поддерживает устройства Electrolux и AEG

После успешного входа в систему приложение Homey опросит API OCP и определит, какие устройства у вас настроены.

Руководство по установке

Для работы этого приложения Homey необходимо, чтобы ваши устройства Electrolux или AEG были настроены на европейский центр обработки данных, так как API сервисы / ассортимент продуктов не работают за пределами этого региона. Это приложение было протестировано с установленной Великобританией в приложении iOS Mobile AEG, но должно хорошо работать по всей территории ЕС.

1. У вас должны быть настроены пароль и электронная почта в вашем мобильном приложении Electrolux или AEG (убедитесь, что вы вышли из системы и вошли снова с использованием электронной почты и пароля, а не OTP).
2. Установите Homey App и настройте параметры с использованием вашей электронной почты и пароля. Эти учетные данные сохраняются, но используются только для создания JWT токена, который многократно используется и обновляется по мере необходимости приложением.
3. Добавьте устройство с помощью приложения и выберите соответствующий тип (прачечная, очиститель воздуха и т.д.).
4. Если ваше устройство не доступно, пожалуйста, посетите https://github.com/gslender/com.gslender.electrolux-aeg.ocp/issues/new/choose, чтобы запросить поддержку для вашего устройства.
